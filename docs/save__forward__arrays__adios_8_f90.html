subroutine save_forward_arrays_adios ( )

Write selected forward arrays in an ADIOS file.

This subroutine is only used for forward simulations when SAVE_FORWARD is set to .true. It dumps the same arrays than save_intermediate_forward_arrays_adios() except than some arrays are only dumped if ROTATION and ATTENUATION are set to .true.
